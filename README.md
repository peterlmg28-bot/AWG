AWG Lookup Table

The awg_data dictionary contains detailed specifications for American Wire Gauge (AWG) sizes from 0000 to 46. For each AWG size, it stores the following properties:

Diameter (mm and inch): The outer diameter of the wire.

Cross-sectional area (mm²): The area of the conductor.

Resistance (Ω/km): Electrical resistance per kilometer.

Normal current (A): Recommended current for continuous operation.

Maximum current (A): Maximum safe current before overheating.
